# Explore wxpy
Explore your wechat using wxpy

## Dependencies
- python 3.6
- jieba  
- matplotlib  
- numpy  
- pandas  
- scipy  
- wordcloud  
- wxpy  

Using `pip` to install what you need.

## Usage
Just type `python wx.py`.

## What you will get
Some info about the sex ratio, province distribution and signature of your wechat friends : )
